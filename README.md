I am a Backend Engineer based in Ho Chi Minh City, Vietnam. I build distributed services and data workflows, and I care about correctness, performance, and operational reliability.

## Selected Impact

- Built production services in Rust and C#/.NET with asynchronous workers, retries/DLQs, and practical observability.
- Shipped warehouse-management APIs in C# ASP.NET MVC + SQL Server supporting 50+ staff, 10,000+ SKUs, and 500+ daily operations.
- Automated Excel bulk updates (1,000+ records per batch), reducing manual data entry by 70% and cutting operational inconsistencies by 85%.
- Improved backend API response time by 40% (850ms to 510ms) through profiling and SQL-heavy query optimization.
- Built a microservices capstone using Spring Boot + React/TypeScript with MySQL, Redis, Firestore, Kafka, CI/CD, and monitoring (Prometheus/Grafana).
